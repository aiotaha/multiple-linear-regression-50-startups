# 📈 Multiple Linear Regression on 50_Startups
Predicting startup profits with Multiple Linear Regression in Python


---

## 📖 Overview

This repository demonstrates how to build a **Multiple Linear Regression** model using the **50_Startups** dataset.

The notebook covers the complete machine learning workflow, from data preprocessing and feature encoding to model training, prediction, and evaluation using Python and Scikit-Learn.

It serves as a beginner-friendly yet practical example for understanding linear regression in supervised machine learning.

---

## 🎯 Project Objectives

- Explore the 50_Startups dataset
- Perform data preprocessing
- Encode categorical variables
- Split data into training and testing sets
- Train a Multiple Linear Regression model
- Predict startup profits
- Evaluate model performance

## 📊 Dataset

The **50_Startups** dataset contains information about startup companies including:

| Feature | Description |
|----------|-------------|
| R&D Spend | Research and Development expenditure |
| Administration | Administrative expenses |
| Marketing Spend | Marketing expenditure |
| State | Startup location |
| Profit | Target variable |

The goal is to predict **Profit** based on the available features.
